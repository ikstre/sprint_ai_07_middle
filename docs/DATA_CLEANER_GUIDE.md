# 데이터 정제 및 통합 가이드 (DATA_CLEANER_GUIDE.md)

이 문서는 데이터셋의 중복을 제거하고 수동 보정치를 반영하여 **최종 정제 파일(cleaned)**을 생성하는 과정을 설명합니다.

### 1. 개요
수집된 로우 데이터의 중복을 제거하고, 직접 검증한 메타데이터를 결합하여 데이터의 **무결성**을 확보하는 것이 목적입니다. 텍스트 정제뿐만 아니라 엑셀 출력 시 발생하는 수식 오류 방지 로직이 포함되어 있습니다.

### 2. 환경 설정 및 경로
스크립트는 실행 환경을 자동으로 감지하여 서버와 로컬 경로를 선택합니다.

* **서버 환경**: **/srv/shared_data/** 하위 경로를 참조합니다.
* **로컬 환경**: 프로젝트 루트의 **../data/** 폴더를 참조합니다.

**필수 폴더 구조**
* **./data/files/**: 원본 PDF 및 HWP 파일 위치
* **./data/data_list.csv**: 원본 데이터 리스트
* **./data/data_list_fixed.csv**: 수동 보정된 메타데이터 파일

### 3. 실행 방법
터미널에서 아래 명령어를 입력하여 전처리를 수행합니다.

```bash
python scripts/data_cleaner.py
```

### 4. 핵심 처리 로직
**데이터 중복 제거**
* 본문 텍스트를 공백 없이 결합하여 **MD5 해시**를 생성하고 대조합니다.
* 내용이 동일할 경우 **발주 기관명 출현 빈도** 및 **문서 내 키워드 위치**를 점수화하여 가장 정확한 원본 파일을 선별합니다.

**수동 데이터 통합 (Fixed Data Update)**
* **파일명**을 고유 식별자(Key)로 사용하여 **data_list_fixed.csv**의 값을 원본에 이식합니다.
* **공고 번호, 사업 금액, 공개 일자, 입찰 참여 시작일, 입찰 참여 마감일**의 5개 핵심 컬럼을 우선적으로 업데이트합니다.

**텍스트 정제 및 안정성**
* 엑셀에서 하이픈(**-**)이나 등호(**=**)로 시작하는 텍스트가 수식으로 오인되는 현상을 방지하기 위해 접두사 처리를 수행합니다.
* 모든 필드에 **quoting=csv.QUOTE_ALL** 설정을 적용하여 본문 내 쉼표에 의한 **열 밀림 현상**을 차단합니다.

### 5. 결과물 확인
* 실행 완료 후 **./data/data_list_cleaned.csv** 파일이 생성됩니다.
* 이 파일은 모든 중복이 제거되고 보정치가 반영된 **최종 데이터셋**이므로, 이후의 분석 및 모델 학습 시 이 파일을 참조해 주시기 바랍니다.

### 6. 유의사항
* **파일명** 변경 시 데이터 매칭이 불가능하므로 원본 파일명을 유지해야 합니다.
* 새로운 결측치를 발견하여 수정할 경우 **data_list_fixed.csv**에 내용을 추가한 후 스크립트를 재실행하십시오.
